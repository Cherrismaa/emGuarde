## 🧠 About emGuarde

emGuarde is a product-focused informational website created to explain and showcase a **patented Japanese EMF harmonization technology** by Enagic. The product is designed to help **neutralize and divert harmful electromagnetic radiation** emitted by everyday electronic devices such as mobile phones, Wi-Fi routers, laptops, TVs, and home appliances.

The website focuses on:
- Simple, easy-to-understand explanations  
- A clean and calm user experience  
- Trust-building, professional presentation  
- High performance and SEO optimization  

This project also reflects my learning journey as a beginner web developer while following real-world development and deployment practices.

## 🛠 Tech Stack Used

This project is built using modern frontend technologies:

- **React** – Component-based UI development  
- **TypeScript** – Type safety and maintainability  
- **Vite** – Fast development and build tooling  
- **Tailwind CSS** – Utility-first responsive styling  
- **shadcn/ui** – Accessible and clean UI components  

These tools help deliver a smooth, fast, and scalable user experience.
